📖 ما هي المعادلة التربيعية؟

المعادلة التربيعية هي معادلة تكتب بالصورة العامة:

أ س² + ب س + جـ = ٠

حيث أ، ب، جـ أعداد حقيقية و أ ≠ ٠

✨ قيم س التي تحقق المعادلة تسمى حلول (جذور) المعادلة ✨

يوجد ٣ طرق لحلها

اضغط على أي طريقة لمشاهدة الشرح والأمثلة 👇

🔍 الطريقة الأولى: الحل بالتحليل

💡 الفكرة الأساسية

نستخدم خاصية: إذا كان أ × ب = ٠، فإن أ = ٠ أو ب = ٠

يعني: نحول المعادلة إلى ضرب قوسين، ثم نساوي كل قوس بصفر.

📝 خطوات الحل

١اكتب المعادلة على الصورة: أ س² + ب س + جـ = ٠
٢حلل الطرف الأيمن إلى عاملين (قوسين).
٣ساوِ كل قوس بصفر.
٤حل المعادلتين البسيطتين لإيجاد قيم س.
🟢 مثال ١: سهل

حل المعادلة:

س² - ٥ س + ٦ = ٠

🔸 نحلل إلى قوسين:

(س - ٢)(س - ٣) = ٠

🔸 نساوي كل قوس بصفر:

س - ٢ = ٠   أو   س - ٣ = ٠

🔸 نحل لإيجاد س:

✓ الحل: س = ٢   أو   س = ٣
🟡 مثال ٢: متوسط

حل المعادلة:

س² + ٧ س = ٠

🔸 نأخذ س عامل مشترك:

س (س + ٧) = ٠

🔸 نساوي كل قوس بصفر:

س = ٠   أو   س + ٧ = ٠
✓ الحل: س = ٠   أو   س = -٧
🧩 الطريقة الثانية: الحل بإكمال المربع

💡 الفكرة الأساسية

نحول المعادلة إلى صورة (س ± هـ)² = ك، ثم نأخذ الجذر التربيعي لطرفي المعادلة.

تذكر: إذا كان س² = ك، فإن س = ±√ك

📝 خطوات الحل (عندما معامل س² = ١)

١انقل الحد الثابت إلى الطرف الآخر.
٢أوجد (معامل س ÷ ٢)² وأضفه للطرفين.
٣اكتب الطرف الأيمن كمربع كامل (س + معامل س/٢)².
٤خذ الجذر التربيعي للطرفين (± لا تنسى).
٥حل لإيجاد س.
🟢 مثال ١: سهل

حل المعادلة:

س² + ٦ س - ٧ = ٠

🔸 ننقل ٧ للطرف الآخر:

س² + ٦ س = ٧

🔸 نضيف (٦÷٢)² = ٩ للطرفين:

س² + ٦ س + ٩ = ٧ + ٩

🔸 نكتب الأيمن كمربع كامل:

(س + ٣)² = ١٦

🔸 نأخذ الجذر التربيعي:

س + ٣ = ± ٤
✓ الحل: س = ١   أو   س = -٧
🟡 مثال ٢: متوسط

حل المعادلة:

س² - ٤ س - ٥ = ٠

🔸 ننقل ٥ للطرف الآخر:

س² - ٤ س = ٥

🔸 نضيف (٤÷٢)² = ٤ للطرفين:

س² - ٤ س + ٤ = ٥ + ٤

🔸 نكتب الأيمن كمربع كامل:

(س - ٢)² = ٩

🔸 نأخذ الجذر التربيعي:

س - ٢ = ± ٣
✓ الحل: س = ٥   أو   س = -١
⚡ الطريقة الثالثة: القانون العام

🌟 القانون السحري لحل أي معادلة تربيعية 🌟

س = (-ب ± √(ب² - ٤أجـ)) ÷ ٢أ

يستخدم هذا القانون لحل أي معادلة تربيعية مهما كانت!

💡 المميز (Δ)

المميز هو القيمة تحت الجذر: المميز = ب² - ٤ أ جـ

🔸 إذا كان المميز موجب → للمعادلة جذران حقيقيان مختلفان
🔸 إذا كان المميز = صفر → للمعادلة جذر واحد مكرر
🔸 إذا كان المميز سالب → لا يوجد حل حقيقي

📝 خطوات الحل

١اكتب المعادلة على الصورة: أ س² + ب س + جـ = ٠
٢حدد قيم أ، ب، جـ.
٣احسب المميز = ب² - ٤ أ جـ.
٤عوض في القانون العام.
٥احسب الجذرين (± يعني نأخذ مرة + ومرة -).
🟢 مثال ١: سهل

حل المعادلة:

س² + ٥ س + ٤ = ٠

🔸 نحدد: أ = ١، ب = ٥، جـ = ٤

🔸 المميز = (٥)² - ٤(١)(٤) = ٢٥ - ١٦ = ٩

🔸 نعوض في القانون:

س = (-٥ ± √٩) ÷ ٢ = (-٥ ± ٣) ÷ ٢

🔸 نحسب:

س = (-٥+٣)/٢ = -١   أو   س = (-٥-٣)/٢ = -٤
✓ الحل: س = -١   أو   س = -٤
🟡 مثال ٢: متوسط

حل المعادلة:

٢ س² + ٧ س + ٣ = ٠

🔸 نحدد: أ = ٢، ب = ٧، جـ = ٣

🔸 المميز = (٧)² - ٤(٢)(٣) = ٤٩ - ٢٤ = ٢٥

🔸 نعوض في القانون:

س = (-٧ ± √٢٥) ÷ ٤ = (-٧ ± ٥) ÷ ٤

🔸 نحسب:

س = (-٧+٥)/٤ = -٠.٥   أو   س = (-٧-٥)/٤ = -٣
✓ الحل: س = -٠.٥   أو   س = -٣
