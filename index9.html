الدرس الأول: كثيرات الحدود

🎯 ما هي كثيرات الحدود؟

تخيل أنك في ماكينة الصراف الآلي ولديك فئات نقدية مختلفة (1، 5، 10، 20 شيكل). نفس الفكرة في الرياضيات: كثيرة الحدود هي مجموع حدود مختلفة!

💰 المعامل 3
📝 المتغير x
⬆️ الأس 2
🔢 الحد 3x²
انظر كيف نحلل الحد:
3x²
3 المعامل
x المتغير
2 الأس
🏦
قصة: إذا كان عندك 3 أوراق من فئة x² وورقتان من فئة x و5 شيكل نقداً، تكتبها هكذا:
3x² + 2x + 5
هذه هي كثيرة الحدود! 🎯
📌 تعريف كثيرة الحدود
aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀
درجة كثيرة الحدود = أكبر أس في المتغير.
مثال: درجة 2x³ + x² - 5 هي 3 لأن أكبر أس هو 3.

✏️ أمثلة على كثيرات الحدود

مثال 1 3x² + 2x + 5 → درجة 2
مثال 2 x³ - 4x² + x - 7 → درجة 3
مثال 3 2x⁴ + 1 → درجة 4
مثال 4 7 → درجة 0 (ثابت)

🚫 ليست كثيرات حدود

2x⁻¹ + 1 → ❌ الأس سالب
√x + 3 → ❌ الأس كسري (½)
3/x + 5 → ❌ المتغير في المقام

📋 ورقة عمل (1)

س1: ما درجة: 4x³ + 2x² - 5x + 1؟
✅ الدرجة = 3 (لأن أكبر أس هو 3)
س2: ما المعامل في الحد 7x⁵؟
✅ المعامل = 7
س3: كم عدد حدود: 3x² - 2x + 7؟
✅ ثلاثة حدود (3x²، -2x، 7)
س4: هل التالي كثيرة حدود؟ 2x⁻² + 5 (اكتب نعم/لا)
✅ لا، لأن الأس سالب (-2)

الدرس الثاني: الجمع والطرح

➕➖ جمع وطرح كثيرات الحدود

تخيل أن لديك كرتين وقلمين في حقيبة، وأضفت إليها كرة وقلم. كم أصبح المجموع؟ نفس الفكرة هنا!

🍎 2x +
🍎 3x =
🎉 المجموع 5x
🎒
قصة: إذا كان عندك 2x + 3 وأضفت إليها x + 5، فإن المجموع = 3x + 8.
نجمع الـ x مع الـ x، والأرقام مع الأرقام! 🎯
📌 القاعدة الذهبية
نجمع الحدود المتشابهة فقط!
الحدود المتشابهة: لها نفس المتغير ونفس الأس
خطوات الحل:
1️⃣ ضع الحدود المتشابهة معاً
2️⃣ اجمع/اطرح المعاملات فقط
3️⃣ احتفظ بالمتغير والأس كما هو

➕ أمثلة على الجمع

مثال 1 (2x + 3) + (x + 5) = 3x + 8
مثال 2 (5x² + 2x) + (3x² - x) = 8x² + x
مثال 3 (x³ + 4) + (2x³ - 1) = 3x³ + 3

➖ أمثلة على الطرح

مثال 1 (5x + 3) - (2x + 1) = 3x + 2
مثال 2 (7x² + 4x) - (3x² + x) = 4x² + 3x
مثال 3 (4x³ + 6) - (x³ + 2) = 3x³ + 4

⚠️ خطأ شائع

خطأ: 3x + 2x² = 5x³ ❌
الصحيح: 3x + 2x² = 2x² + 3x ✅
(لأن x و x² ليسا حدود متشابهة)

📋 ورقة عمل (2)

س1: أوجد: (3x + 2) + (x + 4) = ؟
✅ 4x + 6
س2: أوجد: (5x² - 2x) + (3x² + x) = ؟
✅ 8x² - x
س3: أوجد: (6a + 5) - (2a + 3) = ؟
✅ 4a + 2
س4: بسّط: (x² + 2x + 1) + (x² - x + 3) = ؟
✅ 2x² + x + 4

الدرس الثالث: الضرب والقسمة

✖️ ضرب وقسمة كثيرات الحدود

هل لاحظت كيف يحسب البقّال السعر؟ إذا كان كيلو الفواكه بـ 3 شيكل، يضرب! نفس الفكرة هنا.

🍎 3 سلات ×
📦 (x+2) =
🎯 الناتج 3x + 6
🍎
قصة: إذا كل سلة فيها (x + 2) تفاحة وأنت اشتريت 3 سلات، فإن مجموع التفاح = 3 × (x + 2) = 3x + 6.
نوزع الـ 3 على كل حد! 🎯
📌 خاصية التوزيع
a × (b + c) = a·b + a·c
📌 قاعدة الأس في الضرب
xᵃ × xᵇ = xᵃ⁺ᵇ
مثال: x² × x³ = x⁵
📌 قاعدة الأس في القسمة
xᵃ ÷ xᵇ = xᵃ⁻ᵇ
مثال: x⁵ ÷ x² = x³
خطوات ضرب حدّين:
1️⃣ اضرب المعاملات (الأرقام)
2️⃣ اجمع الأسس
مثال: 3x² × 4x³ = (3×4) × (x²⁺³) = 12x⁵

✖️ أمثلة على الضرب

مثال 1 2(x + 3) = 2x + 6
مثال 2 x(x + 5) = x² + 5x
مثال 3 3x(2x - 4) = 6x² - 12x
مثال 4 (x + 2)(x + 3) = x² + 5x + 6

➗ أمثلة على القسمة

مثال 1 6x³ ÷ 2x = 3x²
مثال 2 12x⁴ ÷ 4x² = 3x²
مثال 3 8x⁵ ÷ 2x³ = 4x²

📋 ورقة عمل (3)

س1: أوجد: x(2x + 5) = ؟
✅ 2x² + 5x
س2: أوجد: 4x(3x - 2) = ؟
✅ 12x² - 8x
س3: أوجد: (x + 1)(x + 2) = ؟
✅ x² + 3x + 2
س4: أوجد: 10x⁴ ÷ 2x² = ؟
✅ 5x²
