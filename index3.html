🧺 قصة الدرس السريعة:

تخيل أن أحمد معه سلة فيها تفاحة وموزة 🍎🍌، وأخته منى معها سلة فيها موزة وبرتقالة 🍌🍊. لو سألنا: ما الفاكهة المشتركة بينهما؟ ستكون (الموزة) وهذا هو التقاطع! ولو دمجنا السلتين معاً بدون تكرار سنحصل على (تفاحة، موزة، برتقالة) وهذا هو الاتحاد! البساطة كلها هنا! 😉

🔍 مفاهيم أساسية يجب أن نعرفها

1) تقاطع مجموعتين (س ∩ ص) 🤝

هو مجموعة العناصر المشتركة فقط الموجودة في س وفي ص معاً.

شكل فين يوضح التقاطع (المنطقة المشتركة في الوسط)

2) اتحاد مجموعتين (س ∪ ص) 🧺

هو مجموعة جميع العناصر الموجودة في س أو ص أو كلتيهما دون تكرار العنصر.

شكل فين يوضح الاتحاد (تلوين المجموعتين بالكامل)

📊 جدول المقارنة البصري السريع
المفهوم الرمز الكلمة المفتاحية بالمختصر المفيد
التقاطع ∩ و (معاً) المتشابه فقط 👥
الاتحاد ∪ أو (الكل) كل الأرقام بدون تكرار 🧺

💡 تذكر دائماً:

عند كتابة المجموعات، لا تكرر نفس الرقم أبداً داخل القوس { }، فالرياضيات تحب النظام والجمال! وإذا لم يكن هناك أي عنصر مشترك، يكون التقاطع هو المجموعة الخالية ∅ (فاي).

🟢 مثال 1: مستوى سهل (أرقام مباشرة)

إذا كانت س = { ١، ٢، ٣ } ، ص = { ٣، ٤، ٥ } .. أوجد س ∩ ص

١نبحث عن الرقم الموجود في س وفي ص بنفس الوقت.
٢الرقم المشترك الواضح هو الرقم ٣.
الإجابة: س ∩ ص = { ٣ }
🟡 مثال 2: مستوى متوسط (اتحاد مع تكرار)

إذا كانت أ = { ٢، ٤، ٦ } ، ب = { ٦، ٨ } .. أوجد أ ∪ ب

١الاتحاد يعني نجمع كل العناصر في مكان واحد.
٢نلاحظ أن الرقم ٦ متكرر في المجموعتين، نكتبه مرة واحدة فقط!
الإجابة: أ ∪ ب = { ٢، ٤، ٦، ٨ }
🔴 مثال 3: مستوى متقدم (المجموعات الخالية)

إذا كانت ك = { ١، ٣ } ، م = { ٢، ٤ } .. أوجد ك ∩ م

١ك هي مجموعة أرقام فردية، وم هي مجموعة أرقام زوجية.
٢هل يوجد أي رقم مشترك بين الفردي والزوجي؟ لا يوجد أبداً 🛑.
الإجابة: ك ∩ م = ∅ (أو مجموعة خالية {})
⚽ مثال 4: تطبيق حياتي (مجموعات الرياضة من الكتاب)

في حصة الرياضة مع المعلمة نور، اختار الطلاب الألعاب كالآتي:

يوم السبت (س): { سباحة، جري، تنس طاولة }
يوم الإثنين (ص): { تمارين لياقة، كرة قدم }
يوم الأربعاء (ع): { كرة سلة، تمارين لياقة، جري }

المطلوب: أوجد الألعاب المشتركة المتكررة بين يومي السبت والأربعاء (س ∩ ع)؟

١ننظر لقائمة السبت وقائمة الأربعاء ونبحث عن الكلمات المشتركة.
٢نجد أن لعبة (جري) موجودة في اليومين!
الإجابة: س ∩ ع = { جري }
